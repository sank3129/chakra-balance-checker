## 🌟 Features

### 🔮 Comprehensive Chakra Assessment
Utilizing a scientifically-inspired approach, this tool evaluates all seven energy centers:
*   **Root Chakra:** Grounding, security, and survival instincts.
*   **Sacral Chakra:** Creativity, sexuality, and emotions.
*   **Solar Plexus:** Personal power, confidence, and self-esteem.
*   **Heart Chakra:** Love, compassion, and connection.
*   **Throat Chakra:** Communication and self-expression.
*   **Third Eye:** Intuition, insight, and wisdom.
*   **Crown Chakra:** Spiritual connection and higher consciousness.

### 📊 Detailed Analytics Dashboard
Get a deep dive into your spiritual state with:
*   **Energy Balance Score:** A calculated percentage indicating your overall alignment (e.g., 78% Balanced).

*   **Response Pattern Analysis:** Visualizing your tendencies (Rarely, Sometimes, Often, Always).
*   **Emotional Trends:** Tracking your connection to Grounded, Secure, Confident, Connected, Expressive, Intuitive, and Spiritual states.

*   **Communication Boosters:** Tips for imbalanced Throat Chakras, such as joining supportive groups or practicing verbal affirmations.

## 🚀 How to Use

1.  **Start the Assessment:** Answer the 20-question survey regarding your daily life, stress responses, relationships, and spiritual connection.
2.  **Receive Results:** View your overall **Chakra Balance Score** (e.g., 78%).
3.  **Analyze Breakdown:** Review the status of each of the **7 Chakras**. Identify areas that need attention (Imbalanced) versus those that are strong (Balanced).
4.  **Get Recommendations:** Read the **Personalized Insights** section to receive specific steps for restoring harmony to your energy system.
5.  **Export Data:** Save your results and analytics for future reference.

## 💻 Installation & Setup

This is a **static site** requiring no server or database. It works directly in the browser.

### Option A: Direct Usage (Recommended)
1.  Download the `index.html` file.
2.  Double-click to open it in any modern web browser (Chrome, Firefox, Safari, Edge).

### Option B: Deploy with GitHub Pages
1.  Go to your repository settings on GitHub.
2.  Navigate to the **"Pages"** tab.
3.  Under **Branch**, select `main` (or `master`) and the `/ (root)` folder.
4.  Click **Save**. Your app will be live at `https://yourusername.github.io/chakra-analyzer`.

### Option C: Run via Local Server
If you are using VS Code, install the "Live Server" extension and click "Go Live" on `index.html`.
